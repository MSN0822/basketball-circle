# バスケサークル QA テストケース

対象: https://basketball-circle.vercel.app

## 目的

本格始動前に、Vercel本番環境上で主要導線・管理機能・表示品質・後片付け手順を確認する。
テストデータは `TEST_` プレフィックスを付け、実行後に削除する。

## 認証

| ID | シナリオ | 期待結果 |
| --- | --- | --- |
| A-01 | 未ログイン状態でトップページを開く | `/login` にリダイレクトされる |
| A-02 | 有効なテスト会員でログインする | トップページへ遷移し、会員番号と名前が表示される |
| A-03 | 誤ったパスワードでログインする | エラーが表示され、ログイン画面に留まる |
| A-04 | 会員ヘッダーからログアウトする | 未ログイン状態に戻る |
| A-05 | 新規登録タブを開く | 名前、メール、パスワード入力欄が表示される |
| A-06 | 6文字未満のパスワードで登録する | バリデーションエラーが表示される |

| ID | シナリオ | 期待結果 |
| --- | --- | --- |
| E-01 | 公開中イベントがある状態でトップページを開く | 公開中イベントが一覧に表示される |

| J-02 | 未ログインでイベント詳細ページを開く | ログイン画面へ誘導される |
| J-03 | 同じ会員で重複参加する | 重複登録がブロックされる |

## 複数人参加シナリオ

複数人参加、満員、キャンセル待ち、繰り上げの検証は実行レポート側に記録する。
現時点では、キャンセル待ち繰り上げ後に参加番号が重複する不具合が確認されている。
