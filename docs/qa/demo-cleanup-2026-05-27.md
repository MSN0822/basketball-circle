# 運営確認用データ整理 2026-05-27

対象環境: https://basketball-circle.vercel.app

## 目的

運営向けに機能説明しやすいよう、テストで作成したイベントを整理し、機能を網羅したデモイベントだけを残す。

## 実行内容

使用スクリプト:

- `scripts/qa-clean-events-for-demo.mjs`

処理内容:

- 既存イベントを整理。
- 運営確認用デモイベント4件を作成。
- 各イベントに紐づく表示用参加者を作成。
- Authユーザーとmembersレコードは削除していない。

## 作成したデモイベント

| タイトル | ステータス | 用途 |
| --- | --- | --- |
| 【運営確認用】受付中・参加申請デモ | 申請受付中 | 参加申請、キャンセル、友達追加、ニックネーム表示、Google Mapリンク確認 |
| 【運営確認用】満員締切デモ | 締切済み | 定員到達時にキャンセル待ちを作らず締め切る表示確認 |

| 【運営確認用】下書きデモ | 下書き | 管理者画面での下書き表示確認 |

ユーザー側トップページには、下書きを除く3件が表示される。

## 確認観点

- 受付中イベントでは参加申請、キャンセル、友達追加を確認できる。
- 参加者リストで「姓 名(ニックネーム)」と、ニックネームなしの「姓 名」を確認できる。
- 友達は「友達名(招待者の姓の友達)」と臨時IDつきで表示される。
- 満員締切イベントでは定員到達後の締切状態を確認できる。
- 締切日時超過イベントでは、定員未満でも締切状態を確認できる。
- 下書きイベントはユーザー側には出さず、管理者画面で確認できる。
