# 総合テストケース・テストシナリオ 2026-05-26

対象環境: https://basketball-circle.vercel.app

方針:
- 本格始動前にエラー発生確率を下げるため、認証、会員、イベント、参加、友達招待、定員締切、管理者、権限、同時操作、表示を横断的に確認する。
- テストで作成したイベント、会員、参加データは、後から追跡できるように `QA_KEEP_` または `qa_keep_` を含む名称で可能な限り残す。
- 秘密情報はこのファイルに記載しない。
- 自動化できるものはAPI/DB経由で消化し、ブラウザ操作が必要なものは手動確認項目として残す。
